INSTRUCCIONES PARA QUE PUEDAS INSTALAR EL PROYECTO EN TU COMPU:
1: CLONAS EL SIGUIENTE REPOSITORIO(IGUAL YA TE LO COMPARTI CREO GITHUB)
2:CREAS EL ENTORNO VIRTUAL CON: (python -m venv env)
3:ACTIVAS EL ENTORNO VIRTUAL EN WINDOWS CON(.\env\Scripts\Activate.ps1)
4:INSTALAS LAS DEPENDENCIAS CON:(pip install -r requirements.txt)
5:CONFIGURAMOS LA BASE DE DATOS, ABRES MYSQL WORKBENCH Y CREAS UN NUEVO ESQUEMA LLAMADO(Proyecto_Universitario) DESPUES: VAS A SERVER Y DE AHI A DATA IMPORT, Y BUCAS: "Import from Self-Contained File" Y EN LA MISMA CARPETA DEL PROYECTO BUSCAS OTRAS CARPETA LLAMADA "database" Y SELECCIONAS EL SCRIPT QUE VIENE EN ESA CARPETA LE DAD A START IMPORT
6:HACES LAS MIGRACIONES DEL LA BASE DE DATOS A DEJANGO CON(python manage.py migrate)
7:LEVANTAS EL SERVIROR CON(python manage.py runserver)
Y LISTO YA PUEDES ACCEDER A LA PAGINA
